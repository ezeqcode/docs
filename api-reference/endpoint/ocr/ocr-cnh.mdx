description: "Este endpoint realiza o reconhecimento óptico de caracteres (OCR) em imagens de Carteira Nacional de Habilitação (CNH), extraindo informações relevantes como nome, data de nascimento, número de registro, entre outros"
---


<ParamField header="APP-KEY" type="string" placeholder="APP-KEY" required>
  A APP-KEY é essencial para acessar a API. Você pode encontrá-la em [APP TOKEN](https://painel.smink.com.br/apptoken).
</ParamField>


<ParamField body="image" type="file" placeholder="image do cnh (BASE 64)" required>
  Imagem do CNH em base 64
</ParamField>

### Response

<ResponseField name="status" type="string">
  Indica se a chamada foi bem-sucedida. "success" se for bem-sucedida, "error" caso contrário.
</ResponseField>

<ResponseField name="code" type="string">
  O código de status da resposta.
</ResponseField>
