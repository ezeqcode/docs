---
title: "Criar Pagamento"
sidebarTitle: "/payments"
api: "POST https://api.bpix.app/payments"
authMethod: "authorization"
description: "Esse endpoint cria um pagamento com vencimento configurável."
---

<ParamField
  header="Authorization"
  type="string"
  placeholder="Bearer {token}"
  required
>
  O token de autenticação no formato Bearer é obrigatório para acessar este
  endpoint.
</ParamField>

<ParamField
  body="clientMode"
  type="string"
  placeholder="noNeedToSpecify | fillDataNow | requestCheckout"
  required
>
  Define o modo de tratamento dos dados do cliente. 
  - `noNeedToSpecify`: Não exige dados do cliente. 
  - `fillDataNow`: Dados do cliente devem ser fornecidos agora. 
  - `requestCheckout`: Dados do cliente serão preenchidos via checkout.
</ParamField>

<ParamField body="amount" type="number" placeholder="150.50" required>
  Valor do pagamento.
</ParamField>

<ParamField
  body="expire_at"
  type="string"
  placeholder="2025-12-31T23:59:59Z"
  required
>
  Data e hora de expiração no formato ISO 8601.
</ParamField>

<ParamField
  body="description"
  type="string"
  placeholder="Descrição do pagamento"
>
  Descrição do pagamento. Campo opcional.
</ParamField>

<ParamField
  body="external_reference"
  type="string"
  placeholder="Referência externa"
>
  Referência externa à aplicação. Campo opcional.
</ParamField>

<ParamField
  body="redirect_url"
  type="string"
  placeholder="https://sua-url.com/retorno"
>
  URL de redirecionamento após pagamento. Campo opcional.
</ParamField>

<ParamField body="phoneNumber" type="string" placeholder="+5511999999999">
  Número de telefone do cliente. Campo opcional.
</ParamField>

<ParamField body="email" type="string" placeholder="cliente@email.com">
  E-mail do cliente. Campo opcional.
</ParamField>

<ParamField body="document_type" type="string" placeholder="CPF | CNPJ">
  Tipo de documento do cliente. Campo opcional.
</ParamField>

<ParamField body="name_client" type="string" placeholder="Nome do cliente">
  Nome do cliente. Campo opcional.
</ParamField>

<ParamField body="document_client" type="string" placeholder="12345678900">
  Documento do cliente. Campo opcional.
</ParamField>
