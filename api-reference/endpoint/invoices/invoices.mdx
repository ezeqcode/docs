---

<ParamField
  header="Authorization"
  type="string"
  placeholder="Bearer {token}"
  required
>
  O token de autenticação no formato Bearer é obrigatório para acessar este
  endpoint.
</ParamField>

<ParamField body="amount" type="number" placeholder="1.1" required>
  O valor da venda a ser gerada.
</ParamField>

<ParamField body="type" type="string" placeholder="VENDA" required>
  Tipo da transação. Use "VENDA" para indicar uma venda.
</ParamField>

</ResponseField>
