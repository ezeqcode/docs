Para autenticar-se na API da BitPix, você precisa obter um **App Token**. Siga os passos abaixo:

1. Acesse o [painel de controle da BitPix](https://painel.bitpix.com.br).
2. Navegue até a aba **App Token**.
3. Gere e copie o seu **App Token**.

Use esse **App Token** para autenticar suas solicitações à API da BitPix. O **App Token** deve ser incluído no cabeçalho da sua requisição como `APP-KEY`:

```http
APP-KEY: {seu-app-token}
```


## Uso da API

Você pode usar o **App Token** para autenticar-se na API da BitPix. Inclua o **App Token** no cabeçalho da sua requisição como `APP-KEY`.

Lembre-se de que o uso da API está sujeito aos [Termos de Serviço](https://painel.bitpix.com.br/legal) e à [Política de Uso Aceitável](https://painel.bitpix.com.br/legal/policy).
