Para autenticar-se na API da BPIX, você precisa obter um **Chave de API**. Siga os passos abaixo:

1. Acesse o [painel de controle da BPIX](https://painel.bpix.app).
2. Navegue até a aba **Integração**.
3. Gere e copie o seu **Chave de API**.

Use esse **Chave de API** para autenticar suas solicitações à API da BPIX. O **Chave de API** deve ser incluído no cabeçalho da sua requisição em Authorization `Chave de API`:

```http
Chave de API: {sua-chave-api}
```


## Uso da API

Você pode usar o **Chave de API** para autenticar-se na API da BPIX. Inclua o **Chave de API** no cabeçalho da sua requisição como `Chave de API`.

Lembre-se de que o uso da API está sujeito aos [Termos de Serviço](https://bpix.app) e à [Política de Uso Aceitável](https://bpix.app).
