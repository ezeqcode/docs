## Por que liberar IP ou URL?

A liberação de IP ou URL é uma camada essencial de segurança que assegura que apenas servidores autorizados possam acessar nossas APIs. Esse controle protege seus dados e evita acessos não autorizados.

## Quem precisa liberar?

- **Contas Brasileiras:** A liberação de IP ou URL é **obrigatória**.  
- **Contas Uruguaias:** A liberação de IP ou URL é **opcional**, mas altamente recomendada para maior segurança.

## Como liberar IP ou URL?

Realizar a liberação de IP ou URL é simples e rápido. Siga o passo a passo abaixo:

1. **Acesse o Painel de Integração:** Entre no [painel.bpix.app](https://painel.bpix.app).  
2. **Navegue até Integração:** No menu de navegação (navbar), clique em **Integração**.  
3. **Adicione o IP ou URL:** Selecione a opção para adicionar um novo IP ou URL e preencha as informações solicitadas.  
4. **Salve as alterações:** Confirme o cadastro e pronto! Seu IP ou URL estará liberado para uso das APIs.

<Tip>Se precisar liberar múltiplos IPs ou URLs, basta repetir o processo. Qualquer dúvida, estamos prontos para ajudar!</Tip>
